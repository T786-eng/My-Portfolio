# Tohid Shaikh — Portfolio

Built with **Next.js 14 + Tailwind CSS + Framer Motion**. Deployed free on Vercel.

---

## 🚀 Deploy in 10 Minutes (FREE)

### Step 1 — Install Node.js
Download from: https://nodejs.org (choose LTS version)

### Step 2 — Set up the project
Open terminal/command prompt and run:

```bash
# Navigate to your portfolio folder
cd tohid-portfolio

# Install dependencies
npm install

# Test it locally
npm run dev
```

Open http://localhost:3000 — you should see your portfolio!

### Step 3 — Push to GitHub
1. Go to https://github.com → New repository
2. Name it: `portfolio` (or anything)
3. Run these commands:

```bash
git init
git add .
git commit -m "initial portfolio"
git branch -M main
git remote add origin https://github.com/T786-eng/portfolio.git
git push -u origin main
```

### Step 4 — Deploy on Vercel (FREE FOREVER)
1. Go to https://vercel.com
2. Sign up with GitHub (free)
3. Click "New Project"
4. Import your `portfolio` repo
5. Click **Deploy** — done!

Your site will be live at: `https://portfolio-xxxx.vercel.app`

**Custom domain (optional):** In Vercel → Settings → Domains, add a free `.vercel.app` subdomain like `tohidshaikh.vercel.app`

## 📬 Set Up Contact Form (FREE)

1. Go to https://formspree.io
2. Sign up free
3. Create a new form → copy your Form ID
4. Open `src/components/Contact.tsx`
5. Replace `YOUR_FORM_ID` with your actual ID
